# Learning Planner

Design structured learning paths for acquiring new skills or deepening knowledge.

## When to Use

- User wants to learn a new technology/skill
- User asks "how should I learn X?"
- User wants to plan skill development for a goal
- Preparing for a role that requires new skills

## Workflow

### 1. Clarify the Goal

Ask:
- **What:** Specific skill/topic?
- **Why:** Career goal? Project need? Interest? Build on existing skill?
- **Level:** Beginner to expert, or specific proficiency target?
- **Timeline:** Hard deadline or ongoing?
- **Constraints:** Hours per week available?

### 4. Curate Resources

Use `references/resource-guide.md` for evaluation:
- Official docs first (usually best)
- Hands-on courses second (if you learn kinesthetically)
- Books for deep understanding
- Videos for visual concepts
- Projects for integration

**Your preference:** Start with docs + theory, build early, go deeper if it matters

### 5. Set Success Criteria

Path should end with:
- "Can explain X without notes"
- "Can build Y without looking up basics"
- "Can evaluate Z objectively"
- Portfolio project to show what you've learned
